## O que eu construo

- **Análise de Dados e BI:** Desenvolvi um Dashboard de Business Intelligence com dados reais do e-commerce brasileiro Olist, processando mais de 100 mil pedidos e exibindo KPIs e gráficos interativos. Stack: **Python, Pandas, Plotly, Dash**.

- **Arquitetura Back-end e Lógica de Negócios:** Construí a lógica central de um Sistema de Triagem Médica utilizando **Java**, aplicando conceitos de POO, encapsulamento e gerenciamento de estado para garantir uma estrutura escalável.

- **Desenvolvimento Front-end:** Desenvolvi o **Brainlist**, um dashboard pessoal interativo criado do zero com **HTML, CSS e JavaScript**.

- **Automação e Banco de Dados:** Crio rotinas de automação com **Python** e modelo bancos de dados relacionais com **SQL**.
